# Netflix Data Analysis (EDA) Project

## Project Overview
This project analyzes Netflix Movies and TV Shows data using Python, Pandas, NumPy, Matplotlib, and Seaborn.

## Dataset
- Source: Kaggle Netflix Dataset
- Total Records: 8,807
- Movies: 6,126
- TV Shows: 2,664

## Objectives
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Content Distribution Analysis
- Country-wise Analysis
- Rating Analysis
- Release Year Trends

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Netflix content consists of approximately 70% Movies.
- United States has the highest content count.
- TV-MA is the most common content rating.
- Most content was released after 2015.
